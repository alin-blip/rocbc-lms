# Authenticity Policy — ROCBC

**Version:** 1.0  
**Effective from:** [date]

## 1. Purpose

To ensure all submitted work is the learner's own and to provide Pearson with auditable evidence of authenticity verification.

## 2. Learner Identity Verification

Before accessing any course content, every learner must:
1. Upload a valid government-issued ID (passport, national ID, driving licence)
2. Submit a live selfie/video
3. Be approved by admin/IV
4. Accept all policies digitally

## 3. Submission Authenticity

Each submission requires:
- Digital declaration: "I confirm this work is my own"
- IP address + user-agent captured at submission time
- Timestamp recorded 

## 4. Verification Methods

ROCBC uses the following methods to verify authenticity:

a) **Plagiarism detection** — automated screening via Turnitin/Copyleaks
b) **AI detection** — AI-generated content score review
c) **Oral questioning** — for flagged submissions, assessor conducts a Zoom Q&A
d) **Video presentation** — where required by assessment plan
e) **Learning analytics** — login patterns, time on platform

## 5. Oral Questioning

For any submission where authenticity is in doubt:
1. Assessor schedules a 10-15 minute Zoom call
2. Questions probe understanding of the submitted work
3. Session is recorded and stored in evidence vault
4. Notes are added to assessment feedback

## 6. Records

All authenticity verification records (ID, declarations, questioning notes, video) are stored in the evidence vault for 5 years minimum.
